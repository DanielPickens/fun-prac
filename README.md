# fun-prac

Practice stuff.  Nothing major.

This solution solves the below algorithm:


Given a matrix of integers, find the submatrix with largest sum.

Input:
[
  [ 1,  -9, -10,  1],
  [-1,  10,  10,  1],
  [ 0,   9,   9, -9],
  [-1,  -1,  -1, -1]
]
Output: 38
Explanation: Submatrix [[10, 10], [9, 9]]
